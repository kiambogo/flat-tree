# flat-tree

[![build status][1]][2]

API for operating on a binary tree as an array.

Adapted from https://github.com/datrs/flat-tree.

## Installation

``` sh
go get github.com/kiambogo/flat-tree
```

## Usage

``` go
import github.com/kiambogo/flat-tree

flattree.Index(0, 0) // 0
flattree.Depth(0) // 0
```
